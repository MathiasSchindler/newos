# RG

## NAME

rg - short command name for ripgrep

## SYNOPSIS

```
rg [OPTIONS] PATTERN [PATH ...]
rg --files [OPTIONS] [PATH ...]
```

## DESCRIPTION

`rg` is the short entry point for `ripgrep`. It accepts the same options and behavior as `ripgrep`.
